# IOTSwitch Project

To Control Switch Device

## Hardware V0.5

| Pin | Target   | Announce     |
|-----|----------|--------------|
| P4  | NTC_R3   | NAN          |
| P5  | CURR_SEN | NAN          |
| P6  | NTC_R1   | NAN          |
| P7  | TOUCH    | NAN          |
| P8  | WS2812   | NAN          |
| P13 | BUZZER   | NAN          |
| P16 | OUT_PORT | NAN          |
| P17 | OUT_PORT | NAN          |
| P18 | RELAY    | HIGH TO OPEN |
| P2  | MIC_SD   | NAN          |
| P41 | MIC_SCK  | NAN          |
| P42 | MIC_WS   | NAN          |

## Feature
* MQTT control device
* Touchpad control
* Voice control
